# Getup ✊

Webサイト制作のための個人的ボイラープレート

## コマンド一覧

### 開発用タスク
```bash
npm run start
```

### 開発用タスク（IE11 向けに Babel によるトランスパイル工程を追加する場合）
```bash
npm run start-babel
```

### プロダクションファイルのビルド

```bash
npm run build
```

### プロダクションファイルのビルド（IE11 向けに Babel によるトランスパイル工程を追加する場合）

```bash
npm run build-babel
```
